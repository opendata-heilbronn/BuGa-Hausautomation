# Farmbot MySensors Bridge
Control MySensors nodes from FarmBot digital I/O.

## Tasks
- control covers and read cover state
- pump on/off
- read emergency button state 
- tell farmbot to come inside

## Pinout
| Pin  | Function   |
| ---- | ---------- |
| PA08 | RS485 D/E  |
| PA09 | RS485 TX   |
| PA10 | RS485 RX   |
